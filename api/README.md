# ExpressJS API

### MySQL

**MySQL 8.x.x** is the databases Manager.

### env
`.env` file the `api` root directory.

```env
DB_HOSTNAME=127.0.0.1
DB_USER=christian80gabi
DB_PASSWORD=db4free.net
DATABASE=mini_erp

SERVER_PORT=5000
```

### docker-compose up

Run `docker compose up` under the `./api` directory. 


### npm start

```shell
npm install

npm start
```
